Website’s main features:

Seamless Car Rental Experience:
 Users can easily browse available cars, view detailed information, and book rentals for specific dates.

Provider Management System: 
Car owners can list their vehicles, update details, and manage bookings directly from their dashboard.

Real-Time Availability Tracking:
 Each car’s availability status automatically updates when booked to prevent double bookings.

Secure Authentication: 
Firebase Authentication ensures safe login, registration, and Google sign-in with protected private routes.

Interactive & Modern UI:
 Beautiful, responsive interface with SweetAlert and Toast notifications, animated sections using Framer Motion, and a clean car rental theme.
